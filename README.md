# YouTube Analytics CLI

CLI-приложение для построения отчётов по CSV-файлам с метриками YouTube-видео.

Пример запуска:

```bash
python main.py --files stats1.csv stats2.csv --report clickbait
```

Пример запуска тестов:

```bash
python -m pytest
```

Примеры запуска линтера и форматтера:

```bash
ruff check .
```

```bash
ruff format .
```

Для ревью:

- входные CSV-файлы читаются стандартной библиотекой через csv;
- разбор аргументов сделан на argparse;
- вывод таблицы реализован через tabulate;
- архитектура расширяема: новый отчёт добавляется отдельным классом и регистрируется в youtube_analytics/reports/registry.py;
- обработаны пользовательские ошибки для несуществующих файлов и неизвестного имени отчёта;
- критичный функционал покрыт pytest-тестами;
- для линтинга и форматирования используется ruff с конфигурацией в pyproject.toml.
